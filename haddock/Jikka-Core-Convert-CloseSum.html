run :: (MonadAlpha m, MonadError Error m) => Program -> m Program Source #

run reduces summations and products.

  • This doen't do nothing about Foldl.

Examples

Before:

foldl (fun y x -> y + x) 0 (range n)

After:

x * (x - 1) / 2
