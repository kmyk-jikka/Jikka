embedDir :: FilePath -> Q Exp Source #

embedDir find files recursively and embed their contents, like https://hackage.haskell.org/package/file-embed file-embed>'s @embedDir@.

Usage

myDir :: [(FilePath, Data.Text.Text)]
myDir = $(embedDir "dirName")
