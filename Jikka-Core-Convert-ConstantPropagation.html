run :: MonadError Error m => Program -> m Program Source #

run does constant propagation. This assumes that the program is alpha-converted.

For example, this converts the following:

let x = 1
in let f = fun y -> y
in x + x + f(x)

to:

let f = fun y -> y
in 1 + 1 + f(1)

NOTE: this doesn't constant folding.
